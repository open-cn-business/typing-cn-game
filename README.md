我在教孩子练习打字的时候，寻找了一个游戏：https://www.typing.com/student/game/keyboard-jump

但是这个游戏对于小学生来讲并不完美。
- 英文单词不认识，但是国外的小朋友认识
- 只能练习打字，不能充分接受教育

这个游戏的趣味性还是可以的，孩子很专注，受该游戏启发，用 VUE3 做了这个面向中文用户的小游戏，目前还不完善，没有游戏场景。
# 目标

该产品完成以下目标:

- 练习打字
- 练习拼音
- 学习中文词语
